# Workshop Announcement Email

## Subject

AI-Assisted Julia for Scientific Computing Research: Two-Day Hands-On Workshop

## Email Body

Dear colleagues,

We are pleased to announce a two-day, hands-on workshop on **AI-assisted Julia for scientific computing research**. The workshop is designed for **junior faculty and graduate students in mathematics and related quantitative research areas** who already have a working foundation in basic programming concepts.

The workshop will run as **two sessions total, one 2-hour session on each day**, and will focus on a practical workflow for using Codex with Julia in research settings. We will move through three stages: first, setup and workflow discipline; second, a toy Newton root-finding example to practice the AI-assisted coding loop; and third, a paper-based reproduction of **Experiment 1** that extends into a benchmark comparison. The emphasis will be on careful implementation, validation, and documentation rather than broad coverage.

Participants should plan to **bring their own laptops**. Before the workshop begins, they must already have:

- **Codex installed and working** (the Julia coding assistant we will use in the workshop)
- **Julia installed and working**
- comfort with basic programming concepts such as variables, functions, loops, and arrays

Because the workshop is hands-on, we will not spend core teaching time on installing the required tools. The expectation is that everyone arrives ready to run the workshop material from the start of Session 1.

The goal of the workshop is to help participants build a disciplined workflow for small, testable Julia code units, paper-grounded reproduction work, and cautious extension into benchmarking. We will be explicit about assumptions, validation, and the limits of what can be claimed from a reproduction, especially around the paper's Experiment 1 result.

If you are interested in attending, please make sure your laptop is ready in advance and that your Codex and Julia setup has been verified.

Best regards,  
The workshop organizers
